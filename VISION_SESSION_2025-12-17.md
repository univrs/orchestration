## Executive Summary

This document captures the philosophical and technical vision for Univrs.io's distributed infrastructure protocol. We are **not** building a multi-cloud abstraction layer that reinforces Big Cloud dependency. We **are** building a sovereign, peer-to-peer infrastructure protocol where:

- Individual data and preferences are cryptographically protected
- Trust is earned through reputation, not purchased from vendors
- Resources flow through mycelial credit economics
- The system is designed from day one to operate without centralized cloud providers

### 4.2 Command Structure

```bash
ui - univrs interface (universal interface)

IDENTITY COMMANDS
  ui init                    # Generate new identity (keypair)
  ui identity show           # Display your public identity
  ui identity export         # Export identity for backup (encrypted)
  ui identity import         # Import identity from backup
  ui identity rotate         # Rotate keys (advanced)

POLICY COMMANDS  
  ui policy show             # Display current policy
  ui policy set              # Interactive policy editor
  ui policy trust <operator> # Add operator to trust list
  ui policy deny <operator>  # Add operator to deny list
  ui policy limits           # Set resource/spending limits

NODE COMMANDS (for operators)
  ui node init               # Initialize this machine as a node
  ui node register           # Register node with network
  ui node status             # Show node status
  ui node offer              # Configure resources to offer
  ui node withdraw           # Remove node from network

WORKLOAD COMMANDS (for users)
  ui deploy <spec>           # Deploy a workload
  ui scale <name> <n>        # Scale workload replicas
  ui status                  # Show all your workloads
  ui logs <name>             # Stream workload logs
  ui delete <name>           # Remove workload

NETWORK COMMANDS
  ui network status          # Show network overview
  ui network peers           # List connected peers
  ui network discover        # Discover new nodes

CREDITS COMMANDS
  ui credits balance         # Show your credit balance
  ui credits history         # Show transaction history
  ui credits send <to> <amt> # Transfer credits (if policy allows)

ADMIN COMMANDS
  ui config show             # Show configuration
```
